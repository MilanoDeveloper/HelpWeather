# HelpWeather 🌦️
Aplicativo Android em Kotlin para exibir informações climáticas usando OpenWeatherMap API.

## 🧰 Tecnologias
- Kotlin
- Jetpack Compose
- Retrofit
- Room Database
- Google Maps API

## 🚀 Como rodar
1. Clone o repositório
2. Configure sua API key no arquivo `local.properties`
3. Rode o app via Android Studio
